# Portfolio Web - Desarrollador DAW

Portfolio web profesional creado con HTML, CSS, Bootstrap y JavaScript para mostrar habilidades y proyectos como desarrollador web.


- Diseño Responsive daptado para todos los dispositivos

- Secciones Incluidas:
  - Hero/Inicio con presentación
  - Sobre Mí
  - Habilidades Técnicas (Frontend, Backend, Base de Datos)
  - Proyectos con tarjetas interactivas
  - Formulario de Contacto
  - Enlaces a Redes Sociales

- Efectos Visuales:
  - Animaciones al hacer scroll
  - Efectos hover en tarjetas
  - Navegación suave entre secciones
  - Navbar que cambia al hacer scroll


## Tecnologías Utilizadas

- **HTML5**: Estructura semántica
- **CSS3**: Estilos personalizados y animaciones
- **Bootstrap 5.3.2**: Framework CSS para diseño responsive
- **Bootstrap Icons**: Iconos
- **JavaScript**: Interactividad y efectos
